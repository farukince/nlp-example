# Türkçe RAG Asistan

Bu proje, Türkçe dokümanlar üzerinden kaynaklı cevap üreten mini bir RAG asistanıdır.

## Amaç

- Türkçe PDF dokümanlarını okumak
- Metinleri parçalamak
- Embedding üretmek
- Qdrant vector database içine kaydetmek
- Kullanıcı sorusuna kaynaklı cevap üretmek

## Teknolojiler

- Python
- Qdrant
- Sentence Transformers
- Ollama
- Streamlit
- FastAPI
